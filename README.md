# Machine Learning Algorithm Comparison

## Project Overview

This project focuses on comparing the performance of multiple machine learning algorithms on a single dataset. Different classification algorithms were applied to the Breast Cancer dataset, and their accuracies were compared to identify the best-performing model.

## Algorithms Used

- Logistic Regression
- Decision Tree
- Random Forest
- K-Nearest Neighbors (KNN)

## Dataset Used

- Breast Cancer Dataset
(Source: Scikit-learn built-in dataset)

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Project Task

Apply multiple machine learning algorithms on one dataset and compare their performance using accuracy scores.

## Results

The algorithms were trained and tested on the Breast Cancer dataset. Accuracy scores were calculated for each algorithm and compared using tables and graphs.


## Conclusion

The project helped in understanding how different machine learning algorithms perform on the same dataset. Random Forest and Logistic Regression achieved higher accuracy compared to other algorithms.
